# Arranger Lab 1.0.0 implementation report

Date: 2026-07-17

## Delivered

- Native macOS SwiftUI application built with CoreMIDI, AVFoundation/CoreAudio and Accelerate.
- Manufacturer-independent actions, profiles and drivers with PA700 firmware 1.5.0 as the first physical vertical.
- Unique-ID endpoint discovery, hot-plug, MIDI input/output, monitor filters, capture, replay, diff and mandatory Panic.
- Canonical `.arrlab` packages with raw JSONL events, analysis, short WAV evidence and CSV/SMF compatibility exports.
- Audio analysis for RMS, peak, normalized spectrum, centroid and spectral distance.
- Expert-mode model challenge, destination and byte visibility, arbitrary SysEx confirmation and automatic expiry.
- Complete official PA700 catalogues: 1,727 sounds, 379 factory Styles and 298 factory Keyboard Sets.
- Verified operational PA700 identity, part volume/expression/pan/damper, exact presets, transports, SongBook, Keyboard Sets, Arranger Elements and contextual controls.
- Performance scenes, ordered set lists, Show Mode and a deterministic exact-name musical intent preview.
- Isolated AIArranger evaluation at commit `5e72ee6830efffc8c08cddb66fdcafdffcb42a8b`, without modification, redistribution or source reuse.

## Verification

- Dependency-free Arranger Lab harness: 153 checks passed.
- Parser coverage includes running status, fragmented SysEx and interleaved realtime messages.
- CoreMIDI virtual endpoint coverage includes input, output, Unique ID, queue rejection, hot-plug removal and Panic behavior.
- Audio coverage includes synthetic metrics and the PA700 volume and preset evidence rules.
- Error coverage includes invalid profiles, out-of-range values, corrupt packages, denied microphone paths, expired Expert mode and unknown SysEx replay exclusion.
- Release app is ad-hoc signed and verified with `codesign --verify --deep --strict`.
- AIArranger sandbox passed its headless build, 81 tests, JUCE 8.0.4 GUI build, physical chord input, style playback, Panic and clean shutdown.
- Final PA700 smoke test passed with `Jimmy Organ com Brush Ballad na variação 3`; the expected timbre, Style and Variation were physically confirmed and Panic left no stuck notes.

## PA700 acceptance evidence

- Universal identity reply confirms Korg manufacturer `42`, family `0060`, model `005D` and firmware `1.5.0`.
- Right/layer 1 volume uses exact CC7 bytes with strictly increasing RMS and an 8.85 dB total difference.
- Classic Piano is Verified as `CC0=121, CC32=4, PC=0` with A-B-A spectral and panel confirmation.
- Arranger transport is Verified with 120 BPM Clock, Start/Stop, recorded audio, Panic and restoration to Internal clock.
- SongBook entry `9000 - ArrangerLab Test` is Verified with channel-16 NRPN/Data Entry bytes and panel confirmation.
- All documented operational mappings promoted in the PA700 profile are Verified; laboratory discoveries remain Draft until they meet the evidence policy.

## Release boundary

Arranger Lab 1.0.0 is an internal, ad-hoc-signed macOS application. It does not include semantic/generative AI, operational MIDI 2.0, public notarization or physically verified non-PA700 drivers. Those are later-release concerns and are not v1 acceptance items.
