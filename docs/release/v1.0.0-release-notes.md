# Arranger Lab 1.0.0

Arranger Lab 1.0.0 is the first accepted internal release of the manufacturer-independent macOS arranger keyboard laboratory. The Korg PA700 on firmware 1.5.0 is its first physically verified driver.

## Highlights

- Native SwiftUI/CoreMIDI Explorer, send, capture, replay, diff and short audio evidence workflows.
- Manufacturer-independent actions with strict Draft/Verified separation.
- Complete PA700 documentation catalogues: 1,727 sounds, 379 factory Styles and 298 factory Keyboard Sets.
- Verified PA700 mixer, exact preset, transport, SongBook, Arranger Element and contextual control mappings.
- Performance scenes, ordered set lists and focused Show Mode.
- Exact-name local musical assistant with explicit before/after preview and confirmation before MIDI output.
- Mandatory Panic cleanup on stop, disconnect, replay completion, failure and app close.

## Validation

- 153 Arranger Lab harness checks passed after a clean build.
- Release app passed strict ad-hoc code-sign verification.
- ZIP integrity and SHA-256 checksum passed.
- Final PA700 smoke test passed with Jimmy Organ, Brush Ballad, Variation 3 and no stuck notes after Panic.
- AIArranger remained an isolated pinned sandbox; no source or assets were reused.
